Controle de Frotas
Sistema web de controle de frotas tem por objetivo informatizar todo o trabalho de alocação de veiculos e motorista do colégio politecnico da Universidade Federal de santa maria. A primeira versão do sistema web está em funcionamento, desenvolvido na linguagem Java com algumas tecnologias para desenvolvimento de sistemas web, tais como: JavaScript, Jquery, Bootstrap, CSS e para o gerenciamento do dados o software PostgreSQL.

O sistema foi desenvolvido no projeto de pesquisa intitulado Sensibilidade ao Contexto em Sistemas para Ambientas Inteligentes.
